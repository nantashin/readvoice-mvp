# READ VOICE Pro MVP
## Purpose
AI voice web app for visually impaired users. All features usable by voice only.
## Stack
Next.js 14, TypeScript, Tailwind, LangChain ChatOllama, exaone3.5:2.4b, Web Speech API ko-KR, Pretendard Variable CDN
## Colors
primary #0284C7, secondary #0D9488, bg #EBF5FF, text #1E3A5F
## Rules
No any types. All LLM calls via lib/llm/index.ts only. All UI needs aria-label. Claude fallback: claude-haiku-4-5.
