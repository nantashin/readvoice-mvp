# 근무 시간 기록

일일 보고서에 포함될 근무 시간 정보를 기록하는 폴더입니다.

## 사용 방법

### 1. 근무 시간 입력

```powershell
.\scripts\work-time-tracker.ps1
```

대화형으로 다음 정보를 입력:
- **서버 켠 시간** (예: 09:30)
- **서버 끈 시간** (예: 20:00)
- **업무 시작 시간** (예: 09:35)
- **업무 마감 시간** (예: 19:45)

### 2. 자동 포함

입력한 근무 시간은 `daily-report.ps1`이 자동으로 읽어서 일일 보고서에 포함합니다.

```markdown
## 근무 시간

### 수동 기록
- 서버 켠 시간: 09:30
- 서버 끈 시간: 20:00
- 업무 시작: 09:35
- 업무 마감: 19:45

### Git 활동 시간
- 첫 커밋: 09:40
- 마지막 커밋: 19:50
- 총 커밋: 5개
```

## 파일 형식

각 날짜별로 JSON 파일이 생성됩니다:

```json
{
  "date": "2026-04-28",
  "serverStart": "09:30",
  "serverEnd": "20:00",
  "workStart": "09:35",
  "workEnd": "19:45",
  "recordedAt": "2026-04-28 17:30:00"
}
```

## 팁

- 매일 업무 시작 시 `work-time-tracker.ps1`을 실행하여 서버 켠 시간과 업무 시작 시간을 기록하세요.
- 업무 종료 시 다시 실행하여 서버 끈 시간과 업무 마감 시간을 업데이트하세요.
- 비워두려면 Enter를 눌러 "미입력"으로 남길 수 있습니다.
