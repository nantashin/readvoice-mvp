  💡 왜 계속 빼먹는 것처럼 보였나?

  문제 원인

  1. 대화 히스토리 분리 - 각 계정은 독립적인 히스토리를 가짐
  2. 컨텍스트 미전달 - "이어서 해줘"만 말하면 Claude는 이전 상황을 모름
  3. Git 미동기화 - 코드 변경사항이 push되지 않으면 새 계정에서 못 봄

  해결 방법

  1. Git으로 코드 동기화 ← 이제 자동화됨
  2. Handoff 문서로 상황 전달 ← save-state.md
  3. 명시적 첫 메시지 ← 템플릿 제공
