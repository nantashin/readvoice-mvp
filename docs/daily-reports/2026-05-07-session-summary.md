### 2. TTS 속도 시스템

#### 변경사항
- 기본 속도: 1.0 → **2.0배속**
- 범위: 0.5 ~ 10.0 (0.5단계)
- 자연어 명령:
  - "천천히/느리게/빠르잖아" → -0.5배속
  - "빠르게/빨리/너무 느려" → +0.5배속
  - "좋아/그대로/이대로" → 유지
- UI 버튼: 0.5x, 1x, 1.5x, 2x, 2.5x, 3x

### 3. 마이크 효과음

#### lib/audio/mic-sound.ts 신규 생성
```typescript
// 마이크 ON: 띠링~ (상승음)
// 600Hz → 900Hz, 0.15초
playMicOn()

// 마이크 OFF: 띵동~ (하강음)
// 900Hz → 500Hz, 0.2초
playMicOff()
```
- Web Audio API 활용 (외부 파일 없이 코드로 생성)
- 볼륨: 0.3 (작고 경쾌하게)

### 4. BGM 연속 재생 개선

#### 기존 → 신규
```
기존: 분석 시작마다 BGM 재시작 (뚝뚝 끊김)
신규: 앱 시작 시 1회 초기화 → 일시정지/재개 방식
```

#### 핵심 변경
- `stop()` → `pause()` (위치 유지)
- TTS 중: duck() → 볼륨 0.08
- TTS 후: unduck() → 볼륨 0.4 + 자동 재개
- 5곡 자동 순환 (끝나면 다음 곡)

### 7. 이미지 유형별 프롬프트 시스템 (설계 완료)

#### 8가지 유형
1. CHART_WITH_TEXT: 인물관계도, 조직도
2. DOCUMENT_HEAVY: 이력서, 공문서
3. PHOTO_PERSON: 인물사진
4. PHOTO_SCENE: 풍경, 행사사진
5. ILLUSTRATION: 만화, 일러스트
6. SCREENSHOT: 화면 캡처
7. TEXT_ONLY: 텍스트 문서
8. MIXED: 이미지+텍스트 혼합

#### 적용 방법
- gemma4:e2b로 유형 자동 분류 (30초)
- 분류 결과에 맞는 프롬프트 적용
