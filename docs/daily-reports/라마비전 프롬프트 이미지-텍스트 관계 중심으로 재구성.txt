●C✅ 라마비전 프롬프트를 이미지-텍스트 관계 중심으로 재구성했습니다.

  주요 개선사항
                                                                                                                                                               1. 텍스트 역할 파악 시스템
                                                                                                                                                               크기(SIZE)로 판단:
    - Very Large → 제목/주제
    - Large → 소제목/카테고리
    - Medium → 라벨/이름
    - Small → 설명/주석/출처

  위치(POSITION)로 판단:
    - 상단 중앙 → 제목
    - 이미지 옆/위/아래 → 라벨/캡션
    - 하단 → 출처/저작권

  색깔(COLOR)로 판단:
    - 다른 색 → 강조/카테고리 구분

  모양(SHAPE)으로 판단:
    - 볼드체 → 중요 정보
    - 괄호() → 설명/주석
    - 이탤릭 → 인용/대사

  2. 이미지-텍스트 쌍(Pairs) 분석

  각 요소마다:
    • 위치: [어디에]
    • 이미지: [무엇을 보여주는지]
    • 텍스트: [정확한 텍스트]
    • 텍스트 역할: [이름/나이/직책/캡션]
    • 관계: [이 텍스트가 이 이미지를 어떻게 설명하는지]

  3. 구조적 분석 순서

  A. 제목/주제 (가장 큰 텍스트)
  B. 범례/카테고리 (기호 설명)
  C. 이미지-텍스트 쌍 (핵심!)
  D. 연결/관계 표시 (화살표, 선)
  E. 주석/설명 (작은 글씨)
  F. 출처/저작권 (제일 작은 글씨)

  4. 최종 요약 템플릿

  "이 이미지는 [유형]으로, [주제]를 보여줍니다.
  큰 제목 '[제목]'은 [주제]를 나타냅니다.
  이미지들은 [설명]을 보여주고,
  텍스트는 [이름/라벨/관계]를 제공하여,
  전체적으로 [목적]을 이해할 수 있게 합니다."
