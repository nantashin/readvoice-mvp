✅ 주요 완료 사항
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 코드 안정화 (복합 오류 해결)
   - 4월 30일 계정 전환 후 발생한 복합 오류 해결
   - 5af9389 커밋으로 핵심 파일 5개 복구
   - 디버그 상태 표시창 제거 (메뉴:idle, 파일타입:null 등)
   - "시각장애인을 위한" 텍스트 → "AI 음성 도우미"로 변경

2. TTS 속도 시스템 개선
   - 기본 속도 1.0 → 2.0배속으로 변경
   - 속도 범위: 0.5배속 ~ 10.0배속 (0.5단계)
   - 자연어 음성 명령 지원:
     "천천히/느리게" → 0.5배속 감소
     "빠르게/빨리" → 0.5배속 증가
     "좋아/그대로" → 현재 속도 유지
   - 속도 UI 버튼: 0.5x, 1x, 1.5x, 2x, 2.5x, 3x

3. 마이크 ON/OFF 효과음 추가
   - lib/audio/mic-sound.ts 신규 생성
   - Web Audio API 활용 (외부 파일 없이 코드로 생성)
   - 마이크 ON: 띠링~ (상승음, 600→900Hz, 0.15초)
   - 마이크 OFF: 띵동~ (하강음, 900→500Hz, 0.2초)

4. BGM 연속 재생 시스템 개선
   - 기존: 분석마다 BGM 재시작 → 새로 개선: 일시정지/재개 방식
   - 앱 시작 후 BGM 한 번 초기화, 이후 계속 연결 재생
   - 5곡 플레이리스트 자동 순환
   - TTS 중: 덕킹(볼륨 0.08), TTS 후: 볼륨 복구(0.4) + 자동 재개

5. STT 자연어 인식 확장
   - 이미지 관련 키워드 확장 (사진/화면/그림/이미지)
   - 모델 선택 자연어 인식 강화:
     구글 4G: 사/사지/포/뽀 등
     구글 2G: 이/투/이지/투지 등
     큐쓰리: 큐/쓰리/스리/삼/큐쓰리 등
     라마비전: 라/라마/비/비전/람 등
   - GLM 모델 목록에서 제거

6. OCR 후처리 강화
   - 특허 오인식 교정: SHS→특허, SoS→특허청
   - URL/전화번호 패턴 보호
   - 발명자 이름 사이 띄어쓰기 자동 추가
   - DAS 코드 보호 (0246→DAS)
   - Tesseract 초안 문구 사용자에게 노출 금지
   - OCR 교정 프롬프트 개선 (숫자/URL 원본 유지 규칙)
   - pdf-enhance.py 이진화 강화 (300DPI 기준, 대비 2.0배)

7. 이미지 유형별 프롬프트 시스템 설계
   - 8가지 유형 분류: CHART_WITH_TEXT, DOCUMENT_HEAVY,
     PHOTO_PERSON, PHOTO_SCENE, ILLUSTRATION, SCREENSHOT,
     TEXT_ONLY, MIXED
   - gemma4:e2b로 자동 분류 후 유형별 최적 프롬프트 적용
   - (내일 실제 코드 적용 예정)

8. v2.8.0 버전 저장
   - docs/versions/v2.8.0/ 스냅샷 저장 (13개 파일)
   - docs/versions/v2.7.9/ 직전 버전 스냅샷 저장
   - VERSION_HISTORY.md 업데이트
   - 로드맵 PPT 자동 생성
